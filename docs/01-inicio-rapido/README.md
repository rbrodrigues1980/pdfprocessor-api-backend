# 01 — Início Rápido

> Como instalar e rodar o projeto localmente para desenvolvimento e testes.

---

## Documentos nesta seção

| Documento | Descrição |
|-----------|-----------|
| [001 - INSTALACAO_RAPIDA.md](./001%20-%20INSTALACAO_RAPIDA.md) | Guia passo a passo para subir o ambiente local com Docker Desktop. Inclui uso dos scripts `INICIAR.bat`/`PARAR.bat`, conexão com MongoDB Atlas e dicas de troubleshooting rápido. |
| [002 - CONFIG_LOGGING.md](./002%20-%20CONFIG_LOGGING.md) | Como ativar/desativar o salvamento de logs no MongoDB. Logs ficam na collection `logs` com retenção de 30 dias e envio assíncrono. Configuração via `application.yml`. |

---

## Pré-requisitos

- Docker Desktop instalado e rodando
- Java 21 (para desenvolvimento)
- Acesso à string de conexão do MongoDB Atlas

## Ordem de leitura sugerida

1. `001 - INSTALACAO_RAPIDA.md` — Suba o ambiente local
2. `002 - CONFIG_LOGGING.md` — Ajuste os logs conforme necessidade
